# kanban-prosa
Aplikasi ini dapat CRUD task dan juga reorder queue tasks. 
Untuk melakukan delete/update dengan double click pada card task. 

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
